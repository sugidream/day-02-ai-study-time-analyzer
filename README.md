# 🚀 AI Study Time Analyzer

## 📖 Overview

The AI Study Time Analyzer is a Python-based data analysis project developed using NumPy and Jupyter Notebook. This project analyzes weekly study-hour data and generates meaningful insights into productivity, learning consistency, and study habits.

By applying NumPy's powerful array operations and statistical functions, the project transforms raw study-time data into actionable information, helping users understand their learning patterns and monitor progress toward their study goals.

---

## 🎯 Objectives

* Analyze weekly study hours
* Measure overall productivity
* Identify the most productive day
* Identify the least productive day
* Calculate average study time
* Track progress toward a weekly study goal

---

## ✨ Features

* 📊 Total weekly study hour calculation
* 📈 Average daily study hour analysis
* 🔥 Most productive day detection
* 📉 Least productive day identification
* 🎯 Weekly goal progress tracking
* 📋 Clear and organized output presentation

---

## 🛠️ Technologies Used

* NumPy
* Jupyter Notebook

## 📊 Analysis Performed

The notebook performs several analytical operations including:

* Mean calculation

* Percentage-based analysis
* Goal completion tracking
